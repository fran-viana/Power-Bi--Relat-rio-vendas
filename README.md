# Relatório de Vendas e Lucro – Power BI (Projeto DIO)

Este projeto foi desenvolvido como parte de um **bootcamp da Digital Innovation One (DIO)**. Trata-se de um **relatório interativo em Power BI Desktop** para análise de vendas e lucros, com foco em indicadores de desempenho, segmentação de mercado e visualizações geográficas.

##  Visuais utilizados

## Página 1 – Sales Report
- Cartões de indicadores principais:
  - Soma de Vendas (Sales)
  - Unidades Vendidas (Units Sold)
  - Descontos (Discounts)
  - Custo de Mercadoria Vendida (COGS)
  - Lucro Bruto (Gross)
- Gráfico de linhas: Vendas por Mês
- Gráfico de barras: Vendas por Segmento
- Gráfico de barras: Vendas por Produto
- Mapa: Vendas por País

## Página 2 – Report de Lucro Detalhado
- Seleção de Ano (2013 e 2014)
- Tree map: Lucro por Ano e País
- Gráfico de barras: Lucro por Produto
- Tree map: Lucro por Segmento
- Gráfico de colunas: Lucro por Trimestre (com aumento/diminuição)

##  Como visualizar
1. Baixe o arquivo `.pbix` deste repositório.
2. Abra no **Power BI Desktop** (disponível gratuitamente).
3. Use o segmentador de **Ano** para explorar os dados de diferentes períodos.
